> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Implement all four P1 roadmap items: status dashboard, package groups/filtering, `meldr pr`, and hook system.

**Architecture:** Four features sharing a common `PackageFilter` abstraction. Groups/filtering (Task 1-3) must land first since status, pr, and hooks all depend on it. Each feature adds to the `PackageEntry` struct via `#[serde(default)]` for backward compat. All new code follows the existing trait-based pattern (GitOps, TmuxOps) for testability.

**Tech Stack:** Rust 1.88+ (edition 2024), clap 4, serde, toml, rayon, console, thiserror. Tests use assert_cmd, predicates, tempfile. Integration tests run in Docker via `./run-docker-tests.sh`.

**Testing strategy:** Every task includes unit tests AND integration tests. Unit tests cover the core logic in isolation (using mock GitOps/TmuxOps). Integration tests run the actual CLI binary against real git repos in Docker. The existing test pattern is: unit tests in `src/` modules via `#[cfg(test)]`, integration tests in `tests/` via `assert_cmd`.

```rust
use crate::core::workspace::PackageEntry;

#[derive(Debug, Default, Clone)]
pub struct PackageFilter {
    pub only: Vec<String>,
    pub exclude: Vec<String>,
    pub groups: Vec<String>,
}

impl PackageFilter {
    pub fn is_empty(&self) -> bool {
        self.only.is_empty() && self.exclude.is_empty() && self.groups.is_empty()
    }

    pub fn matches(&self, pkg: &PackageEntry) -> bool {
        // --only takes highest precedence
        if !self.only.is_empty() {
            return self.only.iter().any(|o| o == &pkg.name);
        }
        // --group filters by tag
        if !self.groups.is_empty() {
            if !self.groups.iter().any(|g| pkg.groups.contains(g)) {
                return false;
            }
        }
        // --exclude removes packages
        if !self.exclude.is_empty() {
            return !self.exclude.iter().any(|e| e == &pkg.name);
        }
        true
    }

    pub fn apply<'a>(&self, packages: &'a [PackageEntry]) -> Vec<&'a PackageEntry> {
        packages.iter().filter(|pkg| self.matches(pkg)).collect()
    }
}

#[cfg(test)]

            remote: None,

    }

    #[test]
```

```rust
pub enum SyncState {

    pub fn from_divergence(ahead: u32, behind: u32) -> Self {
```

- [ ] **Step 5: Rewrite status display with color-coded table**

Rewrite `src/cli/status.rs` `run()`:

```rust
use crate::core::filter::PackageFilter;
use crate::core::workspace::{self, Manifest};
use crate::core::state::WorkspaceState;
use crate::error::Result;
use crate::git::GitOps;
use console::style;
use std::path::Path;

pub fn run(git: &dyn GitOps, workspace_root: &Path, filter: &PackageFilter) -> Result<()> {
    let manifest = Manifest::load(workspace_root)?;
    let state = WorkspaceState::load(workspace_root)?;

    println!("{} {}", style("Workspace:").bold(), manifest.workspace.name);
    println!();

    // Packages overview
    println!("{}", style("Packages:").bold().underlined());
    let filtered_pkgs = filter.apply(&manifest.packages);
    for pkg in &filtered_pkgs {
        let pkg_path = workspace::package_path(workspace_root, &pkg.name);
        let marker = if pkg_path.exists() {
            style("✓").green()
        } else {
            style("✗").red()
        };
        let groups_info = if pkg.groups.is_empty() {
            String::new()
        } else {
            format!(" [{}]", pkg.groups.join(", "))
        };
        println!("  {} {}{}", marker, pkg.name, style(groups_info).dim());
    }
    println!();

    // Worktree detail table
    if state.worktrees.is_empty() {
        println!("{}", style("No active worktrees.").dim());
        return Ok(());
    }

    for (branch, wt) in &state.worktrees {
        let tmux_info = wt.tmux_window.as_deref()
            .map(|w| format!(" [tmux: {w}]"))
            .unwrap_or_default();
        println!("{} {}{}", style("Worktree:").bold(), branch, style(tmux_info).dim());

        // Table header
        println!("  {:<16} {:<8} {:<12} {:<30} {:<8}",
            style("Package").underlined(),
            style("Status").underlined(),
            style("Ahead/Behind").underlined(),
            style("Last Commit").underlined(),
            style("Sync").underlined(),
        );

        for pkg in &filtered_pkgs {
            let wt_path = workspace::worktree_path(workspace_root, branch, &pkg.name);
            if !wt_path.exists() {
                continue;
            }

            let dirty = git.is_dirty(&wt_path).unwrap_or(false);
            let (ahead, behind) = git.divergence(&wt_path, &format!("origin/{branch}"))
                .unwrap_or((0, 0));
            let last_commit = git.log_oneline(&wt_path, 1)
                .unwrap_or_default()
                .into_iter().next()
                .unwrap_or_else(|| "—".to_string());
            // Truncate commit to 30 chars
            let last_commit = if last_commit.len() > 30 {
                format!("{}…", &last_commit[..29])
            } else {
                last_commit
            };
            let sync_state = SyncState::from_divergence(ahead, behind);

            let status_str = if dirty {
                style("dirty").yellow().to_string()
            } else {
                style("clean").green().to_string()
            };
            let ab_str = format!("↑{} ↓{}", ahead, behind);
            let ab_styled = if behind > 0 {
                style(ab_str).red().to_string()
            } else if ahead > 0 {
                style(ab_str).yellow().to_string()
            } else {
                style(ab_str).green().to_string()
            };
            let sync_str = match sync_state {
                SyncState::Synced => style("synced").green().to_string(),
                SyncState::Stale => style("stale").yellow().to_string(),
                SyncState::Conflict => style("conflict").red().to_string(),
            };

            println!("  {:<16} {:<8} {:<12} {:<30} {:<8}",
                pkg.name, status_str, ab_styled, last_commit, sync_str,
            );
        }
        println!();
    }

    Ok(())
}
```

```rust
use crate::core::workspace::{WorkspaceHooks, PackageEntry, Manifest};
use crate::error::Result;
use std::path::Path;
use std::process::Command;

pub fn resolve_hooks_for_event(
    event: &str,
    ws_hooks: &WorkspaceHooks,
    pkg_hooks: &WorkspaceHooks,
) -> Vec<String> {
    let pkg_cmds = match event {
        "post_sync" => &pkg_hooks.post_sync,
        "post_worktree_create" => &pkg_hooks.post_worktree_create,
        "pre_remove" => &pkg_hooks.pre_remove,
        "post_pr" => &pkg_hooks.post_pr,
        _ => return vec![],
    };
    if !pkg_cmds.is_empty() {
        return pkg_cmds.clone();
    }
    match event {
        "post_sync" => ws_hooks.post_sync.clone(),
        "post_worktree_create" => ws_hooks.post_worktree_create.clone(),
        "pre_remove" => ws_hooks.pre_remove.clone(),
        "post_pr" => ws_hooks.post_pr.clone(),
        _ => vec![],
    }
}

pub fn run_hooks(
    event: &str,
    manifest: &Manifest,
    packages: &[&PackageEntry],
    work_dir_fn: impl Fn(&str) -> std::path::PathBuf,
) {
    use rayon::prelude::*;

    packages.par_iter().for_each(|pkg| {
        let cmds = resolve_hooks_for_event(event, &manifest.hooks, &pkg.hooks);
        if cmds.is_empty() {
            return;
        }
        let dir = work_dir_fn(&pkg.name);
        for cmd in &cmds {
            eprintln!("[hook] {}: running '{}' in {}", pkg.name, cmd, dir.display());
            let result = Command::new("sh")
                .arg("-c")
                .arg(cmd)
                .current_dir(&dir)
                .status();
            match result {
                Ok(status) if status.success() => {}
                Ok(status) => {
                    eprintln!(
                        "[hook] WARNING: '{}' in {} exited with {}",
                        cmd, pkg.name, status
                    );
                }
                Err(e) => {
                    eprintln!(
                        "[hook] WARNING: '{}' in {} failed: {}",
                        cmd, pkg.name, e
                    );
                }
            }
        }
    });
}
```

```rust
fn extract_github_repo(url: &str) -> Option<String> {
    // Handle both HTTPS and SSH formats:
    // https://github.com/fmcevoy/meldr.git -> fmcevoy/meldr
    // git@github.com:fmcevoy/meldr.git -> fmcevoy/meldr
    let url = url.trim_end_matches(".git");
    if let Some(rest) = url.strip_prefix("https://github.com/") {
        Some(rest.to_string())
    } else if let Some(rest) = url.strip_prefix("git@github.com:") {
        Some(rest.to_string())
    } else {
        None
    }
}
```

**Detecting dirty or unpushed packages:**
```rust
fn has_changes(git: &dyn GitOps, path: &Path, branch: &str) -> Result<bool> {
    // Check for uncommitted changes
    if git.is_dirty(path)? {
        return Ok(true);
    }
    // Check for unpushed commits (ahead > 0)
    let (ahead, _) = git.divergence(path, &format!("origin/{}", branch))?;
    Ok(ahead > 0)
}
```

**The `create` function flow:**
```rust
pub fn create(
    git: &dyn GitOps,
    workspace_root: &Path,
    cwd: &Path,
    filter: &PackageFilter,
    title: Option<String>,
    body: Option<String>,
    draft: bool,
) -> Result<()> {
    let manifest = Manifest::load(workspace_root)?;
    let branch = detect_current_worktree_branch(workspace_root, cwd)?;
    let packages = filter.apply(&manifest.packages);

    // Phase 1: Find packages with changes
    let mut dirty_pkgs = Vec::new();
    for pkg in &packages {
        let wt_path = workspace::worktree_path(workspace_root, &branch, &pkg.name);
        if wt_path.exists() && has_changes(git, &wt_path, &branch)? {
            dirty_pkgs.push(pkg);
        }
    }

    if dirty_pkgs.is_empty() {
        println!("No packages have changes to create PRs for.");
        return Ok(());
    }

    // Phase 2: Push and create PRs
    let title = title.unwrap_or_else(|| branch.clone());
    let mut created_prs: Vec<(String, String)> = Vec::new(); // (pkg_name, "owner/repo#num")

    for pkg in &dirty_pkgs {
        let wt_path = workspace::worktree_path(workspace_root, &branch, &pkg.name);
        let repo = extract_github_repo(&pkg.url)
            .ok_or_else(|| MeldrError::Git(format!("Cannot extract GitHub repo from {}", pkg.url)))?;

        // Push
        let push_output = std::process::Command::new("git")
            .args(["push", "-u", "origin", &branch])
            .current_dir(&wt_path)
            .output()?;
        if !push_output.status.success() {
            eprintln!("WARNING: push failed for {}: {}", pkg.name,
                String::from_utf8_lossy(&push_output.stderr));
            continue;
        }

        // Create PR via gh
        let mut gh_args = vec![
            "pr", "create",
            "--title", &title,
            "--repo", &repo,
        ];
        if draft {
            gh_args.push("--draft");
        }
        let pr_body = body.clone().unwrap_or_default();
        gh_args.extend(["--body", &pr_body]);

        let gh_output = std::process::Command::new("gh")
            .args(&gh_args)
            .current_dir(&wt_path)
            .output()?;

        if gh_output.status.success() {
            let url = String::from_utf8_lossy(&gh_output.stdout).trim().to_string();
            // Extract PR number from URL: https://github.com/owner/repo/pull/42 -> 42
            let pr_num = url.rsplit('/').next().unwrap_or("?");
            let pr_ref = format!("{}#{}", repo, pr_num);
            println!("  {} {}", style("✓").green(), style(&pr_ref).bold());
            created_prs.push((pkg.name.clone(), pr_ref));
        } else {
            eprintln!("WARNING: gh pr create failed for {}: {}",
                pkg.name, String::from_utf8_lossy(&gh_output.stderr));
        }
    }

    // Phase 3: Update PR bodies with cross-references
    if created_prs.len() > 1 {
        let xref_body = build_cross_reference_body(&manifest.workspace.name, &created_prs);
        for (pkg_name, _) in &created_prs {
            let wt_path = workspace::worktree_path(workspace_root, &branch, pkg_name);
            let _ = std::process::Command::new("gh")
                .args(["pr", "edit", "--add-body", &xref_body])
                .current_dir(&wt_path)
                .output();
        }
    }

    // Phase 4: Run post_pr hooks
    let pkg_refs: Vec<&PackageEntry> = dirty_pkgs.into_iter().cloned().collect();
    // (hook wiring added in Task 7)

    println!("\nCreated {} PRs.", created_prs.len());
    Ok(())
}
```

**Error handling for partial failures:** If push fails for one package, skip it and continue with others. If `gh` fails, warn and continue. Print a summary at the end showing which PRs were created and which failed. The overall command exits 0 if at least one PR was created.
